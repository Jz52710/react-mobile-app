## 二、快速安装

### 1、clone到本地

`git clone https://github.com/baozouai/react-mobile-app.git`

### 2、打开终端，在项目根目录运行以下命令

`npm install`

### 3、然后运行`npm start`开启项目

项目默认运行在`http://localhost:3000/`，可自行修改端口
